# Use Google Sheets via Custom OAuth 2.0 Python MCP Server

We need a persistent, accessible, and structured database for tracking scraped jobs and application pipelines. We decided to use Google Sheets as the persistent storage engine, interacting with Google Sheets and Google Drive APIs via a custom Python MCP server configured with OAuth 2.0 user credentials. This provides a visual spreadsheet interface in Google Drive while maintaining robust, programmatic CRUD operations for AI agents.
