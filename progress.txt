## Iteration 3 — Issue #26: Batch LLM contact classification (classify_contacts)

Added `classify_contacts(items, settings)` async function to `src/core/outreach.py`.

The function sends a single batch Gemini prompt with all raw Apify profiles formatted as numbered entries. Gemini returns a JSON array of classified contacts with `russian_speaker` and `is_recruiter` flags. Results are capped at 5 per audience type. Contacts matching neither type are excluded. Settings (`OutreachSettings`) control which audience types are included in the output.

Added `import json` to `src/core/outreach.py`.

Added 5 new tests (TDD red→green) to `tests/test_outreach.py`:
